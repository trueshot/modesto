# ModelT Warehouse Infrastructure

Shared utilities and tools for working with ModelT warehouse specifications.

## Structure

- **scripts/** - Node.js utilities for generating and parsing warehouse specs
  - `generate-svg.js` - Generate SVG floor plans from JSON specifications
  - `warehouseParser.js` - Parse ModelT SVG specs and extract 3D data
  - `harvestWarehouseSVG2.js` - Batch process and validate warehouse files
